## 4. Việc còn dở — cần tiếp tục

### 🔴 Ưu tiên cao

**R4 — Confirm modal khi drag card vào Closed Won/Lost:**

State đã sẵn sàng trong `CrmLeadManager.tsx`:
```typescript
const [pendingClosedDrag, setPendingClosedDrag] = useState<{
  leadId: string;
  targetStatus: 'closed_won' | 'closed_lost';
} | null>(null);
```

Còn cần thêm **confirm modal JSX** vào return tree, ngay trước closing `</div>` của component. Ví dụ pattern:

```tsx
{/* Confirm Close Lead */}
{pendingClosedDrag && (
  <div className="fixed inset-0 z-50 flex items-center justify-center ...">
    <div className="...">
      <h3>Xác nhận đóng Lead?</h3>
      <p>Sau khi chuyển sang {pendingClosedDrag.targetStatus === 'closed_won' ? 'Closed Won' : 'Closed Lost'}, lead sẽ bị khóa và không thể drag tiếp.</p>
      <button onClick={() => {
        updateLeadStatus(pendingClosedDrag.leadId, pendingClosedDrag.targetStatus);
        setPendingClosedDrag(null);
      }}>Xác nhận</button>
      <button onClick={() => setPendingClosedDrag(null)}>Hủy</button>
    </div>
  </div>
)}
```

Vị trí inject: sau `{/* Add Lead Modal */}` block, trước `</div>` cuối.

### 🟡 Ưu tiên trung bình

- **KanbanColumn `isLocked` prop chưa được truyền** từ kanban render loop — cần thêm `isLocked={statusKey === 'closed_won' || statusKey === 'closed_lost'}` vào mỗi `<KanbanColumn>`.
- **KanbanCard `isDraggable` prop** được nhận nhưng chưa được dùng để disable drag handle (chỉ ẩn thôi).
- **Build verification:** Chưa chạy `npm run build` sau sprint 2 — nên chạy để đảm bảo không có TypeScript error mới.

## 5. Kiến trúc hiện tại

```
[Neon DB]
   ↕ (REST via fetch)
[Next.js API Routes]
   /api/crm/leads      → CRUD leads + state machine validation
   /api/crm/staff      → CRUD staff members
   /api/crm/permissions → RBAC matrix
   /api/crm/audit-logs → Append-only audit trail
   ↕
[Zustand Store — useCrmStore.ts]  (persisted to localStorage, notifications in-memory)
   ↕
[React Components]
   CrmHeader (notifications bell)
   CrmSidebar (navigation + stats widget)
   CrmLeadManager (table + kanban D&D)
   CrmLeadDetailModal (localStatus sync, locked status)
   CrmRbacManager (team management, RBAC matrix)
   CrmUserSettings (profile — name locked)
   CrmAuditLog (timeline view — super_admin only)
```

**Quyền hạn (RBAC):**
- `super_admin` → full access kể cả Audit Log, re-open closed leads
- `crm_manager` → xem team, quản lý leads, không xem Audit Log
- `sales_rep` → chỉ xem/update leads của mình
